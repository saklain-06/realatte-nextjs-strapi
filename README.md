# Real Estate Project Platform (Next.js + Strapi)

A modern real estate listing platform built with:

- Next.js 16 (App Router)
- React + Tailwind CSS
- Strapi CMS (local backend)
- Dynamic routing
- Project detail pages
- Image gallery (upcoming)
- Lead form integrated with Strapi
- Filters + search (city, builder, status)
